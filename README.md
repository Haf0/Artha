## Artha 
 an Personal finance manager App for Android. and it is a simple and easy to use app to manage your expenses and incomes.
 ## Benefits
    - Easy to use
    - Simple
    - Secure because its offline
    - No ads
    - No in-app purchases
    - No internet permission
    - No analytics
    - No tracking
    - No data collection
    - No data selling
    - No data sharing
    - No data mining
